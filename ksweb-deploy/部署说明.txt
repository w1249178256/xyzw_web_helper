XYZW Token Manager - KSWEB部署说明
===================================

1. 安装KSWEB
   - 在安卓手机上安装KSWEB应用
   - 可从Google Play或官网下载

2. 部署步骤
   a) 将ksweb-deploy文件夹中的所有文件复制到KSWEB的htdocs目录
      默认路径: /storage/emulated/0/ksweb/htdocs/
   
   b) 确保KSWEB设置:
      - 启用Apache服务器
      - 启用mod_rewrite模块（用于URL重写）
      - 端口设置（默认8080）

3. 访问应用
   - 浏览器访问: http://localhost:8080
   - 或局域网访问: http://[手机IP]:8080

4. 注意事项
   - 本应用为纯前端SPA，无需PHP支持
   - 数据存储在浏览器localStorage和IndexedDB中
   - 清除浏览器数据会导致token丢失，请做好备份

5. 文件结构
   ksweb-deploy/
   ├── index.html          # 入口文件
   ├── assets/             # JS/CSS资源
   ├── icons/              # 图标文件
   ├── team/               # 角色头像
   ├── fish/               # 鱼类图片
   ├── box/                # 宝箱图片
   ├── .htaccess           # Apache配置（URL重写）
   └── 其他静态资源

6. 故障排除
   - 如果页面空白，检查.htaccess是否正确复制
   - 如果路由不工作，确保mod_rewrite已启用
   - WebSocket连接需要网络权限，确保KSWEB有网络访问权限
